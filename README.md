# GoExplore Data Analytics Dashboards

This repository contains the setup and dashboards for GoExplore's data analytics project. The aim was to migrate data from Google Sheets to **BigQuery**, create shared datasets, and build interactive **OLAP dashboards** in Looker Studio. The dashboards provide descriptive analysis and KPIs for various aspects of the business.

---

## Project Overview

- **Team:** 4 members  
- **Tools Used:** Google Sheets, BigQuery, Looker Studio  
- **Objectives:**
  - Transfer and centralize company data to BigQuery
  - Build interactive dashboards for employees to monitor trends, KPIs, and performance
  - Provide a scalable, collaborative setup for future analysis

---

## Data Setup

1. **Google Sheets**  
   - Original raw data provided by the company.
   - Used for initial exploration and simple visualizations.

2. **BigQuery**  
   - Centralized repository for all project data.
   - SQL queries used to structure and aggregate data for OLAP dashboards.

3. **Looker Studio**  
   - Connected to BigQuery for live, interactive dashboards.
   - Used filters, scorecards, charts, and tables to make insights accessible.

---

## Project Highlights

- Migrated raw data from Google Sheets to BigQuery.
- Built interactive dashboards using Looker Studio.
- Demonstrated descriptive analysis for KPIs and business insights.
- Established a scalable data setup for future company reporting.

---

## What I Learned

- **Data Warehousing:** How to structure and manage data in BigQuery for analysis.  
- **SQL Queries:** Aggregating, filtering, and preparing data for dashboards.  
- **OLAP Concepts:** Differences between OLTP and OLAP, and how to design dashboards for analytics.  
- **Data Visualization:** Choosing appropriate charts (bar, line, pie, scatter, maps) for different types of data.  
- **Looker Studio Skills:** Connecting data sources, creating interactive dashboards, filters, scorecards, and styling visuals.  
- **Team Collaboration:** Coordinating tasks across multiple dashboards and ensuring consistent data interpretation.  
- **Presentation of Insights:** Making descriptive and actionable data insights accessible to stakeholders.

## Notes

- All dashboards are OLAP-style and provide employees with interactive access to trends, KPIs, and performance metrics.
